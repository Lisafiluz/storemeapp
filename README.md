# storemeapp

Target:
Store for buying products


Backend:
  server: flask
  database: sql-alchemy
  forms: wtfforms
  
Frontend:
  js: jquery
  html: jinja
  
Fuctionallity:
1. Sign up
2. Sign in
3. Sign out
4. Search for a product
5. Buy products
6. Update or delete acccount
7. View my orders
